Gambar pilihan format drive saat instalasi Windows
16. Drive yang akan kita install yaitu drive C yang telah diberi nama Windows 10, jadi silakan klik pada Drive 0 Partition 2: Windows 10, kemudian klik Format 
17. Jika keluar jendela notifikasi Windows setup, klik saja Yes 
18. Tunggu beberapa saat, jika tombol Next yang ada di bawah kanan sudah aktif berarti proses format drive C telah selesai. Klik saja tombol Next tersebut 
19. Memulai proses transfer dan instalasi secara otomatis. 
Gambar proses instalasi Windows 10
20. Setelah semua intalasi selesai maka komputer anda akan melakukan restart otomatis. Perhatikan baik-baik, begitu komputer anda restart, segera cabut flashdisk bootable dari USB komputer anda, dan biarkan Windows instalasi Windows berjalan sendiri.
