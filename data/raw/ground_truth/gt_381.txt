Bahasa Pemrograman

1. Hypertext Markup Language (HTML)
   HTML (Hypertext Markup Language)
adalah suatu format data yang digunakan
untuk membuat dokumen hypertext yang
dapat dieksekusi dari satu platform komputer
ke platform komputer lainnya tanpa perlu
melakukan suatu perubahan apapun dengan
suatu alat tertentu (Bhirawa Anaroga
Nandari, Sukadi:2014)

2. Hypertext Preprocessor (PHP)
   PHP memiliki kepanjangan PHP
Hypertext Preprocessor, merupakan suatu
bahasa pemograman yang difungsikan untuk
membangun suatu website dinamis (I. putu
A. E. Pratama, 2014) Sistem yang dibangun
Menggunakan PHP Versi 5.6.19.

3. MySQL
   MySQL adalah suatu Relation Database
Management System (RDBMS) yang
mendukung database yang terdiri dari
sekumpulan relasi atau table (Nandari &
Sukadi, 2014).

4. Cascading Style Sheat (CSS)
   CSS memiliki kepanjangan Cascading
Style Sheet, merupakan suatu bahasa
pemograman web yang digunakan untuk
mengendalikan dan membangun berbagai
komponen dalam web sehingga tampilan web
akan lebih rapi, terstruktur, dan seragam (I.
putu A. E. Pratama, 2014).

5. JavaScript
   JavaScript adalah bahasa pemrograman
yang sederhana karena bahasa ini tidak dapat
digunakan untuk membuat aplikasi ataupun
appalet. Dengan JavaScript kita dapat
dengan mudah membuat sebuah halaman web

yang interaktif. Program JavaScript
dituliskan pada file HTML (*.htm*.html)
(Nandari & Sukadi, 2014).

3. PERANCANGAN SISTEM

a. Sistem Berjalan (Sistem Lama)
   Sistem lama yang digunakan pada KSP
“Harapan Bersama” Selong dapat dijelaskan
pada Gambar 1. Alur Sistem Lama :

Gambar 1. Alur Sistem Lama

   Dari gambar sistem lama Simpanan pada
KSP “Harapan Bersama" di atas dapat
dijelaskan proses sebagai berikut :
1. Anggota akan melakukan simpanan dan
   penarikan.
2. Anggota yang akan melakukan simpanan
   menyerahkan uang beserta buku
   tabungan kepada petugas lapangan
3. Petugas lapangan menerima uang dan
   buku tabungan milik anggota yang
   melakukan simpanan. Petugas lapangan
   mencatat jumlah uang simpanan ke
   dalam buku tabungan dan lembar daftar
   tabungan.
4. Petugas lapangan memberikan uang
   simpanan anggota dan lembar daftar
   tabungan kepada kasir.
5. Kasir menerima uang simpanan dan
   lembar daftar tabungan dari petugas
   lapangan.
6. Sedangkan anggota yang akan
   melakukan penarikan datang ke koperasi
   untuk mengambil uang simpanan kepada
   kasir dengan syarat kasir menerima buku
   tabungan untuk menjadi bukti penarikan.
