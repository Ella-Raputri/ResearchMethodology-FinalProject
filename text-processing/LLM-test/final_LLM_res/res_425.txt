Keterangan:
σt = Tegangan tarik
σtp = Tegangan tarik puncak
εcr = Regangan tarik puncak
ε = Regangan tarik
n = Koefisien tarik (1, 0.75, 0.5, 0.4)

Persamaan 8 untuk mengetahui nilai tegangan tarik pada saat regangan tarik lebih kecil daripada nilai regangan puncak:

σt = E.ε (8)

Persamaan 9 untuk mengetahui nilai damage parameter tekan:

Dt = -2 Del σcu (9)

Dt = 1- √(σt/σtp) (10)

Keterangan:
Dt = Damage parameter tarik

Persamaan 11 untuk mengetahui nilai regangan plastis tekan:

εp inelastic = ε - (σ/E) (11)

Keterangan:
εp inelastic = Regangan plastis tekan
ε = Regangan tekan

Persamaan 12 untuk mengetahui nilai regangan plastis tarik:

εp elastic = εt - (σt/E) (12)

Keterangan:
εp elastic = Regangan plastis tarik
εt = Regangan tarik
E = Modulus elastisitas
